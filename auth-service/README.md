# 🔐 Auth Service with OTP + Redis + Docker

A Spring Boot based authentication service that provides OTP-based login using Redis and JWT, fully containerized using Docker.

---

## 🚀 Features

- 🔑 OTP-based authentication (Email)
- ⚡ Redis for fast OTP storage (with TTL)
- 🔒 JWT token generation after verification
- 🐳 Dockerized application
- 📦 Docker Compose for orchestration
- 🧾 Global exception handling
- 📊 Logging with SLF4J

---

## 🧱 Tech Stack

- Java 17
- Spring Boot
- Spring Security
- Redis
- Docker & Docker Compose
- JWT (jjwt)
- Lombok
